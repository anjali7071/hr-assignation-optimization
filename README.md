# HR Assignation Optimization System

## Overview
This project focuses on optimizing the assignment of employees to tasks based on skills, availability, and workload constraints.

The objective is to improve resource utilization and support efficient workforce planning using data-driven decision-making.

## Objectives
- Model employees, tasks, and skill requirements
- Define constraints such as availability and workload limits
- Generate optimal task assignments
- Improve efficiency in HR resource allocation

## Tech Stack
- Python
- Pandas
- Optimization techniques (Linear Programming / Heuristics)

## Status
Project completed. Tasks were successfully assigned to employees based on skill match and availability constraints.
## Key Insights
- Tasks were assigned based on matching required skills and available working hours.
- The approach ensures efficient utilization of employee skills while respecting workload limits.
- Such assignment logic can support HR teams in improving workforce planning and operational efficiency.
